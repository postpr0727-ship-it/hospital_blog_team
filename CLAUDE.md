# CLAUDE.md

김포다조은병원 블로그 자동 생성 시스템

## 개요
- **목표**: 네이버 블로그 복붙용 완성 본문 + 의료법 준수 + 병원 홍보 15-20%
- **출력**: `output/[날짜]_[한글제목].md` (본문) + `output/[날짜]_[한글제목]_이미지가이드.md`

## 5단계 프로세스

| 단계 | 에이전트 | 입력 | 출력 |
|------|---------|------|------|
| 1 | Planner | 키워드 | `temp/outline.json` (구조+테마) |
| 2 | Researcher | outline | `temp/research.json` (리서치) |
| 3 | Writer | outline+research | `temp/draft.md` (초안) |
| 4 | Compliance Checker | draft | `output/본문.md` (최종) |
| 5 | Image Layout Designer | outline | `output/이미지가이드.md` |

## 사용법
```
"고혈압 겨울철 관리" 주제로 블로그 글 작성해줘
```

## 핵심 규칙

### 문어체 필수
- ❌ "하세요", "드세요" 등 구어체
- ✅ "합니다", "해야 합니다" 문어체

### 의료법 준수
- ❌ "완치", "최고", "100%", "타 병원보다"
- ✅ "개선에 도움", "개인차 있음", "전문의 상담"

### SEO (네이버 상위 노출)
- 제목: 28-32자, "김포" 앞쪽 15자 내 필수
- 첫 문단: 100자 내, 키워드 3회, 지역명, 병원명, 공감 질문
- 표 2개 필수, 질문형 소제목 3개 이상

### 병원 홍보
- 전체 15-20%, 각 본론 섹션에 검사/치료 안내
- "김포다조은병원에서는..." 자연스럽게 삽입

## 파일 구조
```
agents/          - 5개 에이전트 프롬프트
temp/            - 중간 결과물 (outline, research, draft)
output/          - 최종 결과물 (본문, 이미지가이드)
```

## 네이버 블로그 작업 흐름
1. `output/본문.md` → 네이버 블로그에 복붙
2. `output/이미지가이드.md` → AI 프롬프트로 이미지 생성 (나노바나나/ImageFX)
3. 이미지 삽입 후 발행
