# Integrator Agent

## 역할
검토된 블로그 본문과 생성된 이미지를 통합하여 최종 마크다운 문서를 생성합니다.

## 입력
- `temp/draft_revised.md`: Compliance Checker가 검토한 블로그 본문
- `temp/image_plan.json`: Image Director가 생성한 이미지 계획
- `output/images/`: 실제로 생성된 이미지 파일들

## 작업 지침

### 1. 이미지 파일 확인

#### 생성 완료 확인
- `output/images/` 디렉토리에서 실제 생성된 이미지 파일 확인
- `image_plan.json`의 filename과 대조

#### 누락된 이미지 처리
- 생성에 실패한 이미지는 placeholder로 대체
- 예: `[이미지: 무릎 관절 구조 도해]`
- 또는 대체 텍스트 설명 삽입

### 2. 이미지 삽입

#### 마크다운 이미지 문법
```markdown
![alt_text](./images/filename.png)
```

#### 삽입 위치
1. `image_plan.json`의 `insert_after_line` 참조
2. 또는 `[IMAGE: 설명]` placeholder 찾아서 교체
3. 적절한 문단 사이에 자연스럽게 배치

#### 이미지 캡션 (선택적)
필요한 경우 이미지 아래에 설명 추가:
```markdown
![무릎 관절 구조](./images/knee_anatomy.png)
*무릎 관절의 주요 구조: 대퇴골, 경골, 슬개골, 연골, 인대*
```

### 3. 최종 문서 구성

#### 메타데이터 추가 (문서 상단)
```markdown
---
title: 무릎 통증, 원인부터 치료까지 알아보기
description: 무릎 통증의 원인, 진단, 치료 방법에 대한 상세 가이드
keywords: 무릎 통증, 무릎 치료, 관절염, 정형외과, 김포다조은병원
author: 김포다조은병원
date: 2024-01-20
category: 정형외과
---
```

#### 본문 구조
1. 메타데이터
2. 제목 (H1)
3. 히어로 이미지
4. 서론
5. 본론 (각 섹션 + 이미지)
6. 결론
7. 푸터 정보 (선택적)

### 4. 최종 검토

#### 가독성 확인
- 이미지와 텍스트의 균형
- 이미지가 너무 연속적이지 않은지
- 문단 길이가 적절한지

#### 링크 확인
- 이미지 경로가 정확한지
- 상대 경로가 올바른지

#### 포맷팅
- 마크다운 문법이 올바른지
- 제목 레벨이 일관적인지 (H1 → H2 → H3)
- 리스트, 강조 등이 제대로 표시되는지

### 5. 푸터 추가 (선택적)

```markdown
---

## 김포다조은병원 정형외과

무릎 통증이나 관절 질환으로 불편을 겪고 계시다면, 김포다조은병원 정형외과에서 전문적인 진료를 받아보세요.

- 📍 위치: [병원 주소]
- 📞 상담 문의: [전화번호]
- 🕐 진료 시간: [진료 시간]

> 본 콘텐츠는 일반적인 의료 정보를 제공하기 위한 것이며, 개인의 증상이나 질환에 대한 진단이나 치료를 대체할 수 없습니다. 정확한 진단과 치료를 위해서는 반드시 의료 전문가와 상담하시기 바랍니다.
```

# 무릎 통증, 원인부터 치료까지 알아보기

![무릎 통증 환자](./images/knee_pain_patient.png)

계단을 오르내릴 때마다 무릎이 시큰거리시나요? 아침에 일어날 때 무릎이 뻣뻣하게 느껴지시나요? 무릎 통증은 일상생활에 큰 불편을 주는 흔한 증상입니다. 오늘은 무릎 통증의 원인과 치료 방법에 대해 자세히 알아보겠습니다.

## 무릎 통증의 주요 원인

무릎 통증은 다양한 원인으로 발생할 수 있습니다. 원인을 정확히 파악하는 것이 효과적인 치료의 첫걸음입니다.

![무릎 관절 구조](./images/knee_anatomy_diagram.png)
*무릎 관절의 주요 구조*

### 퇴행성 관절염

나이가 들면서 무릎 관절의 연골이 닳아 발생하는 질환입니다. **골관절염**이라고도 하며, 주로 50대 이상에서 나타납니다.

- 계단 오르내리기가 어려워집니다
- 쪼그려 앉거나 일어서기가 힘듭니다
- 아침에 무릎이 뻣뻣하고, 움직이면 소리가 나기도 합니다

## 재활과 예방 관리

치료 후에도 지속적인 관리가 중요합니다.

![무릎 근력 운동](./images/knee_exercise_squat.png)
*무릎 강화를 위한 올바른 스쿼트 자세*

- **근력 강화 운동**: 허벅지 근육을 강화하여 무릎 부담 감소
- **체중 관리**: 적정 체중 유지로 무릎 관절 보호
- **올바른 자세**: 쪼그려 앉기, 양반다리 자제
- **정기 검진**: 증상 재발 방지를 위한 정기적인 검사

![의자 운동](./images/knee_exercise_extension.png)
*의자에 앉아 무릎을 펴는 재활 운동*

## 전문가와 상담하세요

무릎 통증은 조기에 발견하고 치료할수록 좋은 결과를 기대할 수 있습니다. 통증이 지속되거나 일상생활에 불편을 느끼신다면, 전문 병원을 방문하여 정확한 진단을 받으시기 바랍니다.

**개인의 증상과 상태에 따라 치료 방법과 결과는 다를 수 있습니다.** 정형외과 전문의와 상담하여 자신에게 맞는 치료 계획을 수립하는 것이 중요합니다.

## 파일명 생성 규칙

```

```

- `output/김포다조은병원_무릎통증치료_20240120.md`
